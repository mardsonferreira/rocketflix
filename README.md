<h2>:pencil: How to use</h2>

**Requirements**
 - <a href="https://yarnpkg.com/">Yarn</a>

**Steps**
 - Clone the project and go to the folder: `$ git clone https://github.com/mardsonferreira/rocketflix.git && cd rocketflix`.
 - Install dependencies: `$ yarn`
 - Run app: `$ yarn start`
